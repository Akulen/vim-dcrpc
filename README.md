# Discord Rich Presence for Vim

Because why not?

# Usage

To start:

```
:StartDcrpc
```

To stop:

```
:StopDcrpc
```

To automatically start the plugin everytime Vim is started, put this in your `.vimrc`:

```
let g:dcrpc_autostart = 1
```
